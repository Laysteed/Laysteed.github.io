Теоретические материалы

Модуль 1 «Математика» (задания 1 – 5) — направлен на проверку вычислительных навыков, умения решать уравнения, математические задачи, задания с параметром.

Решение уравнений

Линейное уравнение вида: ax + b = 0

Решение: x = -b/a

Пример: 3x + 6 = 0 → x = -2

Проценты

1% = 1/100 = 0.01

Найти a% от числа b: b × (a/100)

Найти, сколько % составляет a от b: (a/b) × 100%

Пример: Найти 15% от 200: 200 × 0.15 = 30

Геометрические фигуры

Площадь прямоугольника: S = a × b

Площадь квадрата: S = a²

Периметр прямоугольника: P = 2(a + b)

Периметр квадрата: P = 4a

Задачи с параметром

Уравнение с параметром имеет вид: ax = b

При a ≠ 0 уравнение имеет единственное решение: x = b/a

При a = 0 и b = 0 - бесконечно много решений

При a = 0 и b ≠ 0 - нет решений

Представлены задачи по информатике из разделов «Цифровая грамотность», «Теоретические основы информатики» и «Алгоритмизация и основы программирования».

Круги Эйлера-Венна

Графическое представление отношений между множествами

Пример: A = города, B = Ярославль, C = Санкт-Петербург

Ярославль и Санкт-Петербург являются городами, поэтому B ⊂ A и C ⊂ A

Единицы измерения информации

1 байт = 8 бит

1 Кбайт = 1024 байт

1 Мбайт = 1024 Кбайт

Информационный объем текста: количество символов × вес одного символа

Пример: "МАМА МЫЛА РАМУ" (12 символов) при весе символа 8 бит:

12 × 8 = 96 бит = 12 байт

Блок-схемы алгоритмов

  • Действие - прямоугольник
  • Решение - ромб
  • Ввод/вывод - параллелограмм
